# ☕ Simple Café Website

## 📖 Project Description
This project is about designing and developing a small, responsive website for a café.  
It displays basic information such as what the café offers, its menu, and how customers can contact the café.

## 🎯 Project Goals
- Create a simple and clean website  
- Make it work on phones and computers  
- Show café information clearly  
- Help customers find and contact the café  

## 🧩 Pages Included
1. **Home Page** – Welcome message and popular food & drinks  
2. **About Page** – Story of the café and its beliefs  
3. **Menu Page** – Breakfast, lunch, drinks, and desserts  
4. **Contact Page** – Contact form, phone number, and address  

## 🎨 Design Style
- Font: Simple and easy to read  
- Layout: Clean and friendly  
- Colors: Brown, Cream, Green  
- Features: Navigation menu, mobile‑friendly design, footer with social links  

## 🛠 Tools Used
HTML | CSS | JavaScript (basic)
